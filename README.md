## Getting Started

1. **Install dependencies:**

```bash
npm install
```

2. **Run the development server:**

```bash
npm run dev
```

3. **Open [http://localhost:3002](http://localhost:3002)** in your browser.

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Fonts:** Bebas Neue (headings), DM Sans (body)
